# Overzealous Premed

## Architecture
- Static site: HTML + CSS + JS (no framework)
- Single page: index.html, styles.css, script.js
- Mentorship + resume site

## Dev
- Open `index.html` in browser, or use VS Code Live Server extension
- No build step, no dependencies, no Docker

## Rules
- Keep it vanilla. No frameworks, no bundlers.
- Mobile responsive
