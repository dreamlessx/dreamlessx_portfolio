december 2024 - present / ~12 hrs/week / surgical assistant + computational analysis

Two-photon calcium imaging of cross-orientation suppression in macaque primary visual cortex. Investigating how neuronal interactions vary across cortical depth using orthogonal plaids versus linear prediction. Built the full XORI analysis pipeline from ROI extraction to mixed-effects modeling.

4,785 ROIs analyzed
28 cortical depths
140-518 depth range (micrometers)
7.9e-10 mixed-effects p-value

methods

Combining non-human primate neurosurgery with computational analysis to map orientation selectivity across laminar circuits.

imaging

Two-photon and three-photon calcium imaging in awake macaque V1. Visual stimuli consist of oriented gratings and orthogonal plaid patterns presented at multiple spatial frequencies.

  • GCaMP-based calcium indicators
  • 28 imaging depths from 140 to 518 micrometers
  • Spanning superficial through deep cortical layers

roi extraction

Automated region-of-interest extraction using Suite2p with custom integration for primate cortical imaging data.

  • Suite2p pipeline integration
  • 4,785 ROIs across all depths
  • Neuropil correction and signal decontamination

xori analysis

Cross-orientation suppression index (XORI) computed by comparing plaid responses to linear predictions from component grating responses.

  • Orthogonal plaid vs linear prediction comparison
  • XORI < 0 = suppression, XORI > 0 = facilitation
  • Computed per-ROI across all stimulus conditions

statistical modeling

Mixed-effects models to account for nested data structure (ROIs within depths within sessions) and assess how XORI varies with cortical depth.

  • Linear mixed-effects regression (statsmodels)
  • Random intercepts for session and depth
  • Mediation analysis for spatial frequency and bandwidth

results

A laminar gradient in cross-orientation interactions, shifting from facilitation in superficial layers to suppression at depth.

p = 7.9 x 10^-10 Mixed-effects model confirms significant relationship between cortical depth and cross-orientation suppression index. XORI decreases with depth.
~40% variance Spatial frequency mediates approximately 40% of the variance in XORI across cortical depth, suggesting tuning-dependent suppression mechanisms.
~21% variance Orientation bandwidth accounts for roughly 21% of variance, indicating that broadly tuned neurons show different suppression profiles than sharply tuned ones.
laminar shift Superficial layers (140-250 micrometers) show net facilitation on average. Deep layers (400-518 micrometers) show net suppression. The transition is gradual, not binary.
4,785 ROIs Full dataset spans 28 cortical depths with sufficient statistical power to resolve laminar differences in cross-orientation interactions.
